# CarrerTracker-AI-Powered Career Management Platform

It is a full-stack web application that helps students and job seekers manage their career preparation in one place. The platform combines job application tracking, Dsa progress monitoring, resume analysis, and AI-powered career guidance into a single dashboard.

The system includes an AI Resume Coach powered by Ollama that analyzes resumes, suggests improvements, generates project ideas, rewrites weak resume sentences, and provides interview preparation guidance.

## Key Features
-> Dashboard

-> Application Tracker

-> DSA Tracker

-> Resume Manager

-> AI Career Coach

-> User Profile
